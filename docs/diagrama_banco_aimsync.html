AimSync — Diagrama Lógico do Banco de Dados
usuarios
PK
id SERIAL NOT NULL
username VARCHAR(100) NOT NULL UNIQUE
senha VARCHAR(255) NOT NULL
perfil VARCHAR(50) NOT NULL
clientes
PK
id SERIAL NOT NULL
nome VARCHAR(255) NOT NULL
cpf VARCHAR(14) NOT NULL UNIQUE
email VARCHAR(255) NOT NULL
telefone VARCHAR(20)
endereco VARCHAR(255)
produtos
PK
id SERIAL NOT NULL
nome VARCHAR(255) NOT NULL
descricao TEXT
preco DECIMAL(10,2) NOT NULL CHECK (>=0)
quantidade_estoque INT NOT NULL CHECK (>=0)
vendas
PK
id SERIAL NOT NULL
data TIMESTAMP DEFAULT CURRENT_TIMESTAMP
valor_total DECIMAL(10,2) NOT NULL CHECK (>=0)
FK
cliente_id INT NOT NULL
FK
usuario_id INT NOT NULL
itens_venda
PK
id SERIAL NOT NULL
FK
venda_id INT NOT NULL
FK
produto_id INT NOT NULL
quantidade INT NOT NULL CHECK (>0)
valor_unitario DECIMAL(10,2) NOT NULL CHECK (>=0)
Legenda
PK = Primary Key (Chave Primária)
FK = Foreign Key (Chave Estrangeira)
||── = Um (obrigatório)
──>< = Muitos (pé-de-galinha)
1 : N = Um para Muitos
